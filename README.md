# LinkedIn Post Generator

Generate LinkedIn posts in your own personal writing style using Few Shot Learning and LLaMA.

## Live Demo
[Try the app here](https://linkedin-post-generator-hzzwnbqmyvgw8j6mu5wzjc.streamlit.app/)

## What It Does
- Paste 2-3 of your own LinkedIn posts in the sidebar
- App analyzes your writing style automatically (tone, structure, vocabulary)
- Enter your topic and preferred length
- Get 3 generated posts that sound like YOU wrote them

## Tech Stack
- LLaMA 3.1 via Groq
- Langchain
- Few Shot Learning
- Streamlit
